### Gaia存储系统
- Blockstack 存储系统简介

- API 简介

  - putFile()
    - 写入文件
    - 文件大小理论上无限制，如果使用 Blockstack官方提供的 gaiahub.blockstack.org，那么上限为25M
  - 纯Gaia框架是不含文件分享、定向加密的等复杂功能，后续功能会通过Blockstack的其他工具来实现
